# restaurant_list
# 功能描述 (features)：
 1. 可以在首頁看到所有餐廳與它們的簡單資料
 2. 可以再點進去看餐廳的詳細資訊
 3. 搜尋餐廳名稱來找到特定的餐廳
 4. 搜尋餐廳類別來找到特定的餐廳

# 安裝與執行步驟 (installation and execution)
 1. 先確認有安裝Node.js以及npm
 2. 將專案clone
 3. 在終端機輸入 npm install 
 4. 再輸入 npm run dev
 5. 開啟瀏覽器，在網址列輸入localhost:3000，就可以開始操作
 6. 停止，請回到終端機按下Ctrl + C
 
# 軟體版本
 1. Node.js v16.17.0
 2. Express 4.16.4
 3. Express-Handlebars 3.0.0
 4. Bootstrap 5.1.3
